Mainly two types of data type i.e.
1. Primitive data type
2. Non-primitive data type

primitive data types :-
1. Number - both integer and float, double are considered as number
2. String - charater, word is considered as string, but prefer the string declaration using backtick, coz it gives more features than single and double quote
3. Boolean - only two possibilties , true or false
4. Undefined - only one value i.e Undefined
5. Null - almost same as Undefined but js doesn't automatically puts, it's type is Object but can be considered as bug
